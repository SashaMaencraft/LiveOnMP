## ✨ О проекте

**VisualVertexRP** — это уникальный Minecraft RolePlay сервер с глубокой экономикой, политическими механизмами и современным стеклянным (Glassmorphism) веб-интерфейсом.

### Особенности сайта:
- 🔮 **Стеклянный дизайн** — современный UI с эффектом размытия и прозрачности (как в Telegram)
- ⌨️ **Бесконечная печатная машинка** — анимированный текст в стиле ретро-терминала
- 📱 **Полная адаптивность** — корректное отображение на всех устройствах (ПК, планшеты, смартфоны)
- 🍔 **Гамбургер-меню** — удобная навигация на мобильных устройствах
- ⚡ **Быстрая работа** — оптимизированные CSS и JS без потери функциональности

## 📋 История обновлений

### Версия 2.0 — Адаптация под мобильные устройства (18 июня 2026)

#### 🆕 Что добавлено:
- **Гамбургер-меню** для всех страниц — навигация сворачивается в выпадающую панель на телефонах и планшетах
- **Медиа-запросы** для планшетов (769–1024px) и телефонов (до 768px)
- **Адаптивная типографика** — размеры шрифтов используют `clamp()` для плавного изменения
- **Горизонтальный скролл** для таблиц и блоков кода на маленьких экранах
- **Увеличенные области касания** для кнопок и ссылок на мобильных устройствах

#### 🔧 Что исправлено:
- **Навигация** — больше не "запаковывается" на телефонах, все пункты доступны через гамбургер
- **Карточки терминов** — теперь корректно отображаются в один столбец на мобильных
- **Сетки правил** — адаптируются под ширину экрана без горизонтального скролла
- **Таблица шпаргалки** — стала горизонтально прокручиваемой на телефонах
- **Отступы и padding** — уменьшены на мобильных для лучшего использования пространства
- **Футер** — ссылки выстраиваются в колонку на телефонах для удобства нажатия

#### 📱 Что стало:
| Было | Стало |
|------|-------|
| Навигация "запаковывалась" на телефонах | Гамбургер-меню с выпадающим списком |
| Карточки выходили за пределы экрана | Адаптивные сетки с `auto-fit` и `minmax` |
| Мелкие кнопки, неудобные для пальцев | Увеличенные области касания |
| Таблицы разрывали макет | Горизонтальный скролл для таблиц |
| Фиксированные размеры шрифтов | Плавная типографика с `clamp()` |
| Одинаковые отступы на всех экранах | Адаптивные padding для мобильных |
