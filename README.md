## ✨ О проекте

**MP: Life** — это уникальный Minecraft RolePlay сервер с глубокой экономикой, политическими механизмами и современным стеклянным (Glassmorphism) веб-интерфейсом.

### Особенности сайта:
- 🔮 **Стеклянный дизайн** — современный UI с эффектом размытия и прозрачности (как в Telegram)
- ⌨️ **Бесконечная печатная машинка** — анимированный текст в стиле ретро-терминала
- 📱 **Адаптивность** — корректное отображение на всех устройствах (ПК, планшеты, телефоны)
- ⚡ **Быстрая работа** — оптимизированные CSS и JS без потери функциональности
- 🎯 **Эффект масштабирования** — плавное увеличение элементов при наведении
- 🛡️ **Защита авторских прав** — все права защищены, копирование запрещено
